== Description ==

Customers can ask questions about a product from the store owner using the NetWeb Ask A Question plugin, even before they make a purchase. Through the use of this plugin, the owner of the shop enables customers to ask inquiries or pose any queries regarding the product being sold, and the admin will respond with a suitable response.

Features: Pose product-related queries.
* The admin can see the history of conversations on the back end (admin panel).
* The customer's account allows them to view past conversations.
* From email configuration, the admin can set the mail subject and mail content that will be sent when buyer submits query.
* Query Form labels can be customized by the admin to meet specific needs.

After the admin has installed the app, the three divisions listed below will be under the admin's control for configuration settings:

* General Configuration: Here, the admin configures the email address of the store owner on which the admin wishes to receive emails.
* Label Configuration: The admin sets up the content for buttons and form labels for the product description page / Enquiry form in accordance with the requirements.
* Mail Configuration: The admin sets up the mail configuration in accordance with the needs, such as adding a subject line, filling in the body of the message.

Asked Questions: By clicking on this tab, the admin can see a history of all consumer inquiries about any product listed in the store.

New Questions: Through the new question button, the admin will be informed of any new inquiries made.
The NEW QUESTION Button itself will also show the quantity of new questions that have been asked, alerting the admin to the new inquiries from customers right away.


After the application has been set up and installed.

You will see the "ASK QUESTION" button on woocommerce product detail page (logged in user).
Additionally, a new popup modal asking for details to input the customer's name, email address, title and description will appear in front of them when they click the "Ask Question" button.

== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
* Activate the plugin through the Plugins screen (Plugins > Installed Plugins).
* You will find 'NetWeb Ask A Question' menu in your WordPress admin screen.
